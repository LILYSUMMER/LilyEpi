NFC ONE-TAP

앱 설치 없이 NFC 태그를 터치하면 브라우저가 자동으로 열려 Web-AR 콘텐츠를 바로 체험할 수 있습니다.

WEB AR 스토리텔링

현실 배경에 제주 신화, 멸종 위기 야생화, 역사적 인물들이 AR로 등장하여 몰입감 있는 스토리텔링을 제공합니다.

VISITOR DATA DASHBOARD

방문자 수, 체류 시간, 미션 완료율을 실시간으로 집계하여 관광지 운영에 필요한 인사이트를 제공합니다.

작동 방식

간단한 3단계로 제주 관광을 완전히 새로운 경험으로

1

NFC 태그 스캔

관광객이 관광지에 설치된 NFC 태그를 스마트폰으로 터치하면 브라우저가 자동으로 열립니다.

{{image}} - NFC 스캔 이미지
2

AR 미션·포토존 체험

Web-AR을 통해 현실 세계에 가상 콘텐츠가 오버레이되어 인터랙티브한 미션과 포토존을 체험합니다.

{{image}} - AR 체험 이미지
3

리워드 쿠폰 자동 발급

미션 완료 시 자동으로 리워드 쿠폰이 발급되어 재방문을 유도하고 관광지 활성화에 기여합니다.
